# employee-management-database
# Project Description
This project is a simple Employee management database built MySQL. It stores employee information such as name, department, salary and joining date.

# Concepts Used
- Table Creation
- Primary Key & Foreign Key
- INSERT, UPDATE, DELETE
- SELECT, WHERE, ORDER BY
- JOIN
- Arregate Functions

# Database Struture
- Department table
- Employee table

# Learning Outcome
- Understanding of relational database design
- Query execution and data manipulation
- Basic normalization concepts
